# Offrion Landing Page Redesign Walkthrough

The Offrion landing page has been redesigned to provide a more modern, centered, and feature-focused experience. Key improvements include a refactored hero section and a more accessible interactive showcase.

## Key Changes

### 1. Centered Hero Feature Grid
The hero section now features a centered layout with a wide 6-item feature grid. This replaces the previous asymmetric layout and highlights technical performance metrics directly at the top.

- **Metrics Highlighted**: Latency (<80ms), Geo-Spatial Targeting, Security Scanning, Real-time Analytics, Instant Redemption, and Automated Splits.
- **Improved Alignment**: The entire hero section is now perfectly centered for better visual balance.
- **Theme Support**: Colors and borders are theme-aware, ensuring visibility in both light and dark modes.

### 2. Vertical Interactive Showcase
The interactive showcase has been moved to its own full-width section below the hero section.

- **Vertical Tabs**: Desktop views now use left-aligned vertical tabs for easier navigation between API, Merchant, and Partner views.
- **Clean Aesthetic**: Removed old-style glow and shadow effects for a cleaner, more premium look.
- **Full Width**: The showcase now spans the container width, providing more breathing room for content.
